# PHP_web
### 이름 : 쇼핑몰 웹 사이트

### 내용 : 포트폴리오용 개인 쇼핑몰 사이트

### 개발 언어 : PHP,HTML,CSS,javascript, json, 

### 개발 DB : MYsql (Mysql Workbench 8.0)

### 개발 프로그램 : Vscode , xampp
##

## 내부 기능 
- 로그인 , 회원 가입
- 검색 기능
- 장바구니 등록
- 찜하기 기능
- 마이 페이지 (사진 변경)
- 구매 기능
- 장바구니에서 구매 기능
